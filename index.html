Table Tags

Tag Name Description
< table> Table Wrapper element for all HTML tables
< th> Table Head Set of rows defining column headers in table
< tbody> Table Body Set of rows containing actual table data
< tr> Table Row Table row container
< td> Table Data Table Data Container
< tf> Table Foot Set of rows defining footer in table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns td element should span
rowspan Row Span Defines how many rows td element should span

Structure Elements

Syntax Name Description
< header> Header Can hold Nav links; intro content
< nav> Nav Links Can be used inside header or on its own
< main> Main Majority of content (main content)
< Footer> Footer ending content; contains contact info/back to top
< section> Section Defines elements (chapters, headings, etc.)
< article> Article Holds content within section
< aside> Aside Additional info, enhances elements (ie. bibliographies, endnotes, comments, pull-quotes
< figure> Figure Adds media(< img src="">)
< audio> Audio Add sound (< source src="">)
<> Attributes controls, source, autoplay, loop
< embed> embed self closing, can be added to other websites
